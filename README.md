# Guía rápida para correr la app

## Requisitos previos
- Tener instalado Node.js (recomendado v18 o superior)
- Tener instalado npm o yarn
- Tener instalado Expo CLI de forma global (`npm install -g expo-cli`)

## Pasos para iniciar el proyecto

1. **Instalar dependencias**

   ```bash
   npm install
   ```

2. **Iniciar el servidor de desarrollo**

   ```bash
   npx expo start
   # o
   expo start
   ```
